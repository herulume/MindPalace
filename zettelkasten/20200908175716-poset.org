#+title: Poset

- tags :: [[file:20200915195028-binary_relation.org][Binary Relation]]

- Partial ordered set
- A [[file:20200915162733-preorder.org][preorder]] that is antisymmetric
- Set $A$ with a [[file:20200915195028-binary_relation.org][binary relation]] $\leq_A$ such that $\forall a,b,c \in A$:
  - Reflexivity :: $a \leq_A b$
  - Transitivity :: if $a \leq_A b$ and $b \leq_A c$ then $a \leq_A c$
  - Antisymmetry :: if $a \leq_A b$ and $b \leq_A a$ then $a = b$
