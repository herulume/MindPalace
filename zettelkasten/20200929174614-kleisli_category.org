#+title: Kleisli category

- tags :: [[file:../../../.local/share/Trash/files/20200824163944-category_theory.org][Category-Theory]], [[file:20200824184713-category.org][Category]], [[file:20200929171123-monad.org][Monad]]

- Given a [[file:20200929171123-monad.org][monad]] $T$ on some [[file:20200824184713-category.org][category]] $C$

- Objects :: $Ob(C)$
- Morphisms :: $X \to Y$ for every $X \to T(Y)$ in $C$ ($Hom_C(X,TY)$)
- Composition :: $g \circ_T f = \mu \circ Tg \circ f$
- Identity morphism :: $\eta$
