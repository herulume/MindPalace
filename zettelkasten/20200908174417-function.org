#+title: Function

- Let $f$ be a function from set $A$ to set a$B$
   $f : A \to B$

- $f$ is defined in all $A$

- Values of $f$ are in $B$
  $range(f) \subseteq B$

- Functions are univocal, total, and injective [[file:20200915195028-binary_relation.org][(binary)]] [[file:20200908175759-relation.org][relations]]
